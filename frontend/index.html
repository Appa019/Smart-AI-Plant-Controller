Por baixo do capo

Stack completa, do firmware ao browser.

O ESP32-C3 le o sensor capacitivo (ADC 12-bit) e o AHT10 via I2C, e envia tudo por HTTP POST pro servidor a cada 10 segundos. O firmware e C++ puro, com calibracao automatica.

O backend e um unico server.py em FastAPI (~1800 linhas) que faz tudo: auth com JWT, ingestao dos dados do ESP32, historico em SQLite, irrigacao remota, e 3 chamadas diferentes pra OpenAI - Vision pra identificar planta, Responses API pra gerar o pixel art, e Web Search pras frases do dia.

Frontend e JS vanilla com Vite, sem framework. Roda na Vercel com Supabase.

Desenvolvimento

Como Foi Construído

Hardware

ESP32-C3 SuperMini com sensor capacitivo v1.2 e AHT10. Firmware C++ com calibração automática por ADC 12-bit. Bomba submersa 80-120L/h com proteção de cooldown e detecção de reservatório vazio

Backend

FastAPI serverless na Vercel com Supabase PostgreSQL. API REST com JWT, ingestão a cada 3s, histórico e comandos remotos. 3 integrações OpenAI: Vision (identificar planta), DALL-E (gerar pixel art) e Web Search (frases diárias)

Frontend

SPA em JavaScript vanilla com Vite. Dashboard responsivo com gráficos históricos, controle de irrigação remoto e pet interativo. Zero dependência de framework -funciona em qualquer navegador
